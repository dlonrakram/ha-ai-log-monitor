# Changelog

## 1.0.0 — 2026-03-15

- Initial release.
- Daily AI-powered log analysis via Perplexity Sonar API.
- Automatic log collection from HA Core via Supervisor API.
- Configurable notification service, model, schedule, and log limits.
- Noise filtering for known noisy log patterns.
- Detailed reports written to HA system log (optional).
- Runs initial analysis on startup for immediate feedback.
